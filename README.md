# Connect In : Epitech Project

A full stack social platform built as a group school project at Web@cadémie by Epitech, Paris.

---

## About the Project

Connect'In is a binôme school project built at Web@cadémie by Epitech, Paris. The goal
was to design and develop a full stack social platform where users can create accounts,
publish posts, interact with other users, and manage their profiles. This project was
built for educational purposes only.

## Features

- Create an account and log in with a secure authentication system using Laravel Sanctum
- Publish posts and interact via comments and likes
- Edit your profile information including email, password, and identity
- Delete your profile entirely or keep your comments and likes on the platform

## Installation

1. Clone the repository:

    git clone git@github.com:EpitechWebAcademiePromo2027/W-WEB-103-PAR-1-1-connect_in-4.git

2. Navigate into the project folder:

    cd W-WEB-103-PAR-1-1-connect_in-4

3. Start the back-end server (run this inside the Back-End folder):

    ./vendor/bin/sail up

4. Start the front-end server (run this inside the Front-End folder):

    npm run dev
